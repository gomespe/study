Para rodar o gulp, na pasta raiz digitar no terminal:
    gulp test

Para testar o servidor:
    No terminal 1 comando:
        node ./lib/server.js
    No terminal 2 comando:
        node ./lib/clientInteractive.js

push <thing to be pushed>
pop
top       #mesma função que peek
size
print   #funcao para debugar, imprime toda a pilha
